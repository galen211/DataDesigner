`DataDesigner` validates configs, generates in-memory previews, creates persisted datasets, lists configured MCP tools, and exposes default model settings.

For runtime settings passed through `set_run_config()`, see [run_config](/code-reference/config/run-config). For persisted creation results returned by `create()`, see [results](results).

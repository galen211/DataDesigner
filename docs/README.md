# Docs Support Files

User-facing docs prose lives under `fern/`.

This directory only keeps support files still consumed by the Fern docs build:

- `notebook_source/` - tutorial notebook source
- `colab_notebooks/` - committed Colab notebooks linked from Fern tutorial pages
- `scripts/` - notebook generation helpers
- `assets/recipes/` - downloadable recipe scripts linked from Fern pages
